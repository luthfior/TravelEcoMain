# TravelECO
A mobile task management application built with:

- **Kotlin**
- **MVVM Architecture**
- **Firebase** (Authentication, Realtime Database, Storage)
- **Midtrans Sandbox Payment Simulator**
- **Android Studio**
- **Figma**
